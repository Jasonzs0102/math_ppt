第2页

一、整式方程

知识点

定义：

整式方程：方程两边都是整式（只含有未知数的加、减、乘、幂运算）的代数式。

类型：

1. 一元整式方程：如一次方程 ax + b = 0 ，二次方程 ax² + bx + c = 0

2. 高次方程：如 x³ - 8 = 0，x⁴ - 5x² + 4 = 0（高次方程需要特殊解法）

解法：

1. 移项、合并同类项，化为标准形式 axn + bxn-1 + ... + k = 0

2. 因式分解法：如 x3 - 8 = 0 可分解为 (x - 2)(x² + 2x + 4) = 0

3. 换元法：如 x⁴ - 5x² + 4 = 0 令 y = x²，则方程化为 y² - 5y + 4 = 0

4. 公式法：解二次方程可使用公式 x = [-b ± √(b² - 4ac)] / (2a)

第3页

整式方程例题（基础）

例题：解方程 x³ - 8 = 0

解：

步骤1：将方程变形为 x³ = 8

步骤2：因式分解法

x³ - 8 = 0

(x - 2)(x² + 2x + 4) = 0

步骤3：由因式分解可得

x - 2 = 0 或 x² + 2x + 4 = 0

x = 2 或 解判别式 Δ = 4 - 16 = -12 < 0

所以 x² + 2x + 4 = 0 在实数范围内无解

∴ 方程 x³ - 8 = 0 的解为 x = 2

开方法解此题

x³ = 8

x = ∛8 = 2

整式方程例题（提高）

例题：解方程 x⁴ - 5x² + 4 = 0

解：

步骤1：观察方程中 x 的次数都是偶数，可以使用换元法

令 y = x²，则原方程变为：

y² - 5y + 4 = 0

步骤2：解一元二次方程

因式分解：(y - 4)(y - 1) = 0

得：y = 4 或 y = 1

步骤3：代回原变量 x

当 y = 4 时，x² = 4，所以 x = ±2

当 y = 1 时，x² = 1，所以 x = ±1

∴ 方程 x⁴ - 5x² + 4 = 0 的解为 x = -2, -1, 1, 2

二、分式方程

知识点

定义：

分式方程：含有未知数分母的代数方程，如 a/x + b/(x-c) = d。

注意事项：

1. 分母为零时，分式无意义，所以要排除未知数取值使分母为零的情况。

2. 解分式方程必须验根，因为去分母可能产生增根！

解法步骤：

步骤1：找出所有分母，记录分母不能为零的条件

步骤2：通分或两边同乘最简公分母，消去所有分母

步骤3：解所得的整式方程

步骤4：验根（检验所得解是否满足分母不为零的条件，且代入原方程）

步骤5：得出最终解（去除不满足条件的解）

分母不为零很重要！

必须验根！

可能会产生增根！

第6页

分式方程例题

例题：解方程 3/(x-2) = 2/x

解：

步骤1：记录分母不为零的条件

x - 2 ≠ 0 且 x ≠ 0，即 x ≠ 2 且 x ≠ 0

步骤2：两边同乘公分母 x(x-2)

3/(x-2) = 2/x

3/(x-2) · x(x-2) = 2/x · x(x-2)

3x = 2(x-2)

步骤3：解整式方程

3x = 2x - 4

x = -4

步骤4：验根

当 x = -4 时，x ≠ 0 且 x ≠ 2，满足分母不为零的条件

验证解是否正确

将 x = -4 代入原方程左边：

3/((-4)-2) = 3/(-6) = -1/2

将 x = -4 代入原方程右边：

2/(-4) = -1/2

∴ x = -4 为方程的解

第7页

三、无理方程

知识点

定义：

无理方程：含有未知数开方的代数方程，如 √(ax+b) = cx+d。

注意事项：

1. 含有根式的式子，其中被开方的表达式必须大于等于零，这是根式有意义的条件。

2. 平方过程可能引入增根，必须验证解是否满足根式存在条件。

3. 解无理方程必须验根！

解法步骤：

步骤1：将含根式的项移到一边，其他项移到另一边

步骤2：两边平方，消去根号

步骤3：整理得到的方程，可能需要重复步骤1-2（如果还有根号）

步骤4：解整式方程

步骤5：验根（检验：① 根式是否有意义；② 代入原方程是否成立）

平方可能导致增根！

必须验根！

检查根式有意义且方程成立

第8页

无理方程例题

例题：解方程 √(2x+3) = x

解：

步骤1：记录根式的存在条件

2x + 3 ≥ 0，即 x ≥ -3/2

步骤2：两边平方，消去根号

√(2x+3) = x

[√(2x+3)]² = x²

2x + 3 = x²

步骤3：整理为标准形式

x² - 2x - 3 = 0

(x - 3)(x + 1) = 0

x = 3 或 x = -1

验证解是否正确

验证 x = 3:

1. 检查根式是否有意义：

2x + 3 = 2(3) + 3 = 9 > 0 ✓

2. 将 x = 3 代入原方程：

左边：√(2·3+3) = √9 = 3

右边：x = 3

左右相等，x = 3 是方程的解 ✓

验证 x = -1:

1. 检查根式是否有意义：

2x + 3 = 2(-1) + 3 = 1 > 0 ✓

2. 将 x = -1 代入原方程：

左边：√(2·(-1)+3) = √1 = 1

右边：x = -1

1 ≠ -1，x = -1 不是方程的解 ✗

∴ 方程 √(2x+3) = x 的解为 x = 3

第9页

四、二元二次方程组

知识点

定义：

二元二次方程组：含有两个未知数，且至少有一个方程中未知数的最高次数为2的方程组。

如：{x + y = 5; x² + y² = 13}，{xy = 6; x - y = 1}

常见形式：

1. 一次和二次：{x + y = m; x² + y² = n} 或 {x - y = m; x² - y² = n}

2. 乘积和和差：{xy = p; x + y = q} 或 {xy = p; x - y = q}

3. 其他形式：包含分式、无理式或更复杂的组合

主要解法：

1. 代入消元法：从一个方程解出一个未知数，代入另一个方程

2. 加减消元法：通过方程组合消去一个未知数

3. 公式法：利用特殊关系如 (x+y)² = x² + 2xy + y²，(x-y)² = x² - 2xy + y²

4. 换元法：引入新变量简化原方程组

5. 解方程组时注意检验所得的解是否满足原方程组

二元二次方程组例题

例题：解方程组 {
x + y = 5
x² + y² = 13
}

解：（代入消元法）

步骤1：从第一个方程解出一个未知数

由 x + y = 5 得：y = 5 - x

步骤2：将 y = 5 - x 代入第二个方程

x² + y² = 13

x² + (5 - x)² = 13

x² + (25 - 10x + x²) = 13

2x² - 10x + 25 = 13

2x² - 10x + 12 = 0

x² - 5x + 6 = 0

(x - 2)(x - 3) = 0

x = 2 或 x = 3

求解 y 值并得到方程组解

当 x = 2 时：

y = 5 - x = 5 - 2 = 3

验证：2² + 3² = 4 + 9 = 13 ✓

所以 (2, 3) 是方程组的一个解

当 x = 3 时：

y = 5 - x = 5 - 3 = 2

验证：3² + 2² = 9 + 4 = 13 ✓

所以 (3, 2) 是方程组的另一个解

∴ 方程组的解为：(2, 3) 和 (3, 2)

提示：也可以用公式法求解

由 (x+y)² = x² + 2xy + y²，可得 2xy = (x+y)² - (x² + y²) = 5² - 13 = 12

五、列方程（组）解应用题

应用题解题步骤

1. 审题

理解题意，明确已知条件和所求量，分析数量关系

2. 设未知数

选择合适的未知数，通常选择所求的量或容易表示的量

3. 列方程（组）

根据题目条件，用代数式表示各个量，建立等量关系

4. 解方程（组）

使用适当的方法求解方程或方程组

5. 检验与答题

验证解是否合理，回答题目所求问题

注意事项：

1. 单位要统一 2. 设未知数时要明确并记录含义 3. 检验解的实际意义

常见应用题类型

工程问题（工作效率）

行程问题（速度、时间）

配比问题（浓度、比例）

数字关系（和差积商）

几何问题（面积、周长）

例题：甲、乙合作完成一项工作需6天，甲单独做比乙少5天，求两人单独完成各需几天。

解：

步骤1：设甲单独完成工作需要 x 天，则乙需要 x+5 天

步骤2：根据工作效率关系列方程

甲的工作效率为：1/x

乙的工作效率为：1/(x+5)

合作时的效率等于各自效率之和：

1/6 = 1/x + 1/(x+5)

两边同乘 6x(x+5)，得：

x(x+5) = 6(x+5) + 6x

x² + 5x = 6x + 30 + 6x

x² + 5x - 12x - 30 = 0

x² - 7x - 30 = 0

(x-10)(x+3) = 0

x = 10 或 x = -3

因为时间不能为负，舍去 x = -3，所以 x = 10

例题：小明与小红相距90千米，两人同时相向而行。小明速度为4.5千米/时，小红速度为5.5千米/时，求他们多久后相遇？

解：

小明走过的路程：4.5t

小红走过的路程：5.5t

根据相遇时两人走过的总路程等于总距离：

4.5t + 5.5t = 90

步骤3：解方程

10t = 90

t = 9

答：两人9小时后相遇

无理方程例题（提高）

例题：解方程 √(x - 1) + √(x + 3) = 4

解：

步骤1：记录根式的存在条件

要使得方程有意义，需要 x - 1 ≥ 0 且 x + 3 ≥ 0，即 x ≥ 1

步骤2：将一个根式移项，准备平方消去根号

√(x - 1) = 4 - √(x + 3)

步骤3：两边平方

(√(x - 1))² = (4 - √(x + 3))²

x - 1 = 16 - 8√(x + 3) + (x + 3)

x - 1 = 16 - 8√(x + 3) + x + 3

-4 = -8√(x + 3)

√(x + 3) = 1/2

解的验证

步骤4：求解并验证

(√(x + 3))² = (1/2)²

x + 3 = 1/4

x = 1/4 - 3 = -2.75

验证：

当 x = -2.75 时，x < 1

不满足根式存在条件！

方程无解

∴ 方程 √(x - 1) + √(x + 3) = 4 在实数范围内无解

提示：在解无理方程时，由于平方运算可能引入增根，验证解是否满足根式存在条件以及代入原方程验证是否成立非常重要。这个例题展示了我们可能得到的解不满足条件的情况。

第15页

代数方程常见错误与提示

易错点总结与解题技巧

常见错误

1. 分式方程：

忘记验根导致增根错误

例：3/(x-1) = 1，当x = 1时分母为0

2. 无理方程：

平方后未检验根的有效性

例：√x = -2 无解，因为√x ≥ 0

3. 应用题：

单位不统一或等量关系错误

解题技巧

1. 审题细致：

理解题意，明确所求

检查特殊条件限制

2. 解题步骤规范：

设未知数→列方程→解方程→验证

分式、无理方程必须验根！

3. 检查与反思：

检查解是否满足题目实际意义

★ 重要提示：方程变形后的解不一定是原方程的解，必须验证！

尤其对分式方程和无理方程，这点非常重要

解题时保持冷静思考，按步骤规范操作，可以避免大多数常见错误！

第17页

代数方程复习总结

核心知识点总结

整式方程

• 形式：只含有未知数的加、减、乘、幂运算

• 解法：因式分解法、换元法、公式法

分式方程

• 形式：含有未知数分母的代数方程

• 解法：通分去分母，注意验根避免增根

应用题

• 步骤：审题→设未知数→列方程→解方程→检验

• 类型：行程、工程、数字、几何、浓度问题

• 注意：单位统一、理解实际意义

• 技巧：找到关键量，建立等量关系

无理方程

• 形式：含有未知数开方的代数方程

• 解法：平方去根号，必须验根避免增根

二元二次方程组

• 形式：两个未知数，至少一个方程次数为2

• 解法：代入法、加减法、公式法

解题关键

1. 整体思路：逐步化简，分析每步意义

2. 验证观念：方程变形可能引入增根

3. 实际应用：理解问题背景和数学关系

灵活运用各类方程的解法，提高解题能力是关键！
