代数方程概述

本章主要学习四类方程

一、整式方程

\(x^3 - 8 = 0\)

二、分式方程

\(\frac{3}{x-2} = \frac{2}{x}\)

三、无理方程

\(\sqrt{2x+3} = x\)

四、二元二次方程组

\(\begin{cases} x + y = 5 \\ x^2 + y^2 = 13 \end{cases}\)

解题关键：理解每类方程的特点和解法，注意验根避免增根

第2页

一、整式方程

知识点

定义：

整式方程：方程两边都是整式（只含有未知数的加、减、乘、幂运算）的代数式。

类型：

1. 一元整式方程：如一次方程 \(ax + b = 0\) ，二次方程 \(ax^2 + bx + c = 0\)

2. 高次方程：如 \(x^3 - 8 = 0\)，\(x^4 - 5x^2 + 4 = 0\)（高次方程需要特殊解法）

解法：

1. 移项、合并同类项，化为标准形式 \(ax^n + bx^{n-1} + ... + k = 0\)

2. 因式分解法：如 \(x^3 - 8 = 0\) 可分解为 \((x - 2)(x^2 + 2x + 4) = 0\)

3. 换元法：如 \(x^4 - 5x^2 + 4 = 0\) 令 \(y = x^2\)，则方程化为 \(y^2 - 5y + 4 = 0\)

4. 公式法：解二次方程可使用公式 \(x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\)

第3页

整式方程例题（基础）

例题：解方程 \(x^3 - 8 = 0\)

解：

步骤1：将方程变形为 \(x^3 = 8\)

步骤2：因式分解法

\(x^3 - 8 = 0\)

\((x - 2)(x^2 + 2x + 4) = 0\)

步骤3：由因式分解可得

\(x - 2 = 0\) 或 \(x^2 + 2x + 4 = 0\)

\(x = 2\) 或 解判别式 \(\Delta = 4 - 16 = -12 < 0\)

所以 \(x^2 + 2x + 4 = 0\) 在实数范围内无解

∴ 方程 \(x^3 - 8 = 0\) 的解为 \(x = 2\)

开方法解此题

\(x^3 = 8\)

\(x = \sqrt[3]{8} = 2\)

第4页

整式方程例题（提高）

例题：解方程 \(x^4 - 5x^2 + 4 = 0\)

解：

步骤1：观察方程中 x 的次数都是偶数，可以使用换元法

令 \(y = x^2\)，则原方程变为：

\(y^2 - 5y + 4 = 0\)

步骤2：解一元二次方程

因式分解：\((y - 4)(y - 1) = 0\)

得：\(y = 4\) 或 \(y = 1\)

步骤3：代回原变量 x

当 \(y = 4\) 时，\(x^2 = 4\)，所以 \(x = \pm 2\)

当 \(y = 1\) 时，\(x^2 = 1\)，所以 \(x = \pm 1\)

∴ 方程 \(x^4 - 5x^2 + 4 = 0\) 的解为 \(x = -2, -1, 1, 2\)

第5页

二、分式方程

知识点

定义：

分式方程：含有未知数分母的代数方程，如 \(\frac{a}{x} + \frac{b}{x-c} = d\)

注意事项：

1. 分母为零时，分式无意义，所以要排除未知数取值使分母为零的情况。

2. 解分式方程必须验根，因为去分母可能产生增根！

解法步骤：

步骤1：找出所有分母，记录分母不能为零的条件

步骤2：通分或两边同乘最简公分母，消去所有分母

步骤3：解所得的整式方程

步骤4：验根（检验所得解是否满足分母不为零的条件，且代入原方程）

步骤5：得出最终解（去除不满足条件的解）

分母不为零很重要！

必须验根！

可能会产生增根！

第6页

分式方程例题

例题：解方程 \(\frac{3}{x-2} = \frac{2}{x}\)

解：

步骤1：记录分母不为零的条件

\(x - 2 \neq 0\) 且 \(x \neq 0\)，即 \(x \neq 2\) 且 \(x \neq 0\)

步骤2：两边同乘公分母 \(x(x-2)\)

\(\frac{3}{x-2} = \frac{2}{x}\)

\(\frac{3}{x-2} \cdot x(x-2) = \frac{2}{x} \cdot x(x-2)\)

\(3x = 2(x-2)\)

步骤3：解整式方程

\(3x = 2x - 4\)

\(x = -4\)

步骤4：验根

当 \(x = -4\) 时，\(x \neq 0\) 且 \(x \neq 2\)，满足分母不为零的条件

验证解是否正确

将 \(x = -4\) 代入原方程左边：

\(\frac{3}{(-4)-2} = \frac{3}{-6} = -\frac{1}{2}\)

将 \(x = -4\) 代入原方程右边：

\(\frac{2}{-4} = -\frac{1}{2}\)

∴ \(x = -4\) 为方程的解

第7页

三、无理方程

知识点

定义：

无理方程：含有未知数开方的代数方程，如 \(\sqrt{ax+b} = cx+d\)

注意事项：

1. 含有根式的式子，其中被开方的表达式必须大于等于零，这是根式有意义的条件。

2. 平方过程可能引入增根，必须验证解是否满足根式存在条件。

3. 解无理方程必须验根！

解法步骤：

步骤1：将含根式的项移到一边，其他项移到另一边

步骤2：两边平方，消去根号

步骤3：整理得到的方程，可能需要重复步骤1-2（如果还有根号）

步骤4：解整式方程

步骤5：验根（检验：① 根式是否有意义；② 代入原方程是否成立）

平方可能导致增根！

必须验根！

检查根式有意义且方程成立

第8页

无理方程例题

例题：解方程 \(\sqrt{2x+3} = x\)

解：

步骤1：记录根式的存在条件

\(2x + 3 \geq 0\)，即 \(x \geq -\frac{3}{2}\)

步骤2：两边平方，消去根号

\(\sqrt{2x+3} = x\)

\([\sqrt{2x+3}]^2 = x^2\)

\(2x + 3 = x^2\)

步骤3：整理为标准形式

\(x^2 - 2x - 3 = 0\)

\((x - 3)(x + 1) = 0\)

\(x = 3\) 或 \(x = -1\)

验证解是否正确

验证 \(x = 3\):

1. 检查根式是否有意义：

\(2x + 3 = 2(3) + 3 = 9 > 0\) ✓

2. 将 \(x = 3\) 代入原方程：

左边：\(\sqrt{2\cdot3+3} = \sqrt{9} = 3\)

右边：\(x = 3\)

左右相等，\(x = 3\) 是方程的解 ✓

验证 \(x = -1\):

1. 检查根式是否有意义：

\(2x + 3 = 2(-1) + 3 = 1 > 0\) ✓

2. 将 \(x = -1\) 代入原方程：

左边：\(\sqrt{2\cdot(-1)+3} = \sqrt{1} = 1\)

右边：\(x = -1\)

\(1 \neq -1\)，\(x = -1\) 不是方程的解 ✗

∴ 方程 \(\sqrt{2x+3} = x\) 的解为 \(x = 3\)

第9页

代数方程复习总结

核心知识点总结

整式方程

• 形式：只含有未知数的加、减、乘、幂运算

• 解法：因式分解法、换元法、公式法

分式方程

• 形式：含有未知数分母的代数方程

• 解法：通分去分母，注意验根避免增根

无理方程

• 形式：含有未知数开方的代数方程

• 解法：平方去根号，必须验根避免增根

二元二次方程组

• 形式：两个未知数，至少一个方程次数为2

• 解法：代入法、加减法、公式法

解题关键

1. 整体思路：逐步化简，分析每步意义

2. 验证观念：方程变形可能引入增根

3. 实际应用：理解问题背景和数学关系

灵活运用各类方程的解法，提高解题能力是关键！
